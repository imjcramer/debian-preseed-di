Helper guides live in this directory.

The installer always stages this index after the target identity exists, then
adds only the guides that match the helpers enabled for the selected host role
and class set. If a helper is not installed, its guide is not copied here.

Current guide names:

- `gitlab-runner.md`
- `nft-policy-generate.md`
- `podbin.md`
- `podbin-service-bridge.md`
